Guidelines for the Image
A. Background :

1. Plain background such as white, black, etc.

2. Back-ground and reference/package color should be different.

B. Object positioning :

1. Package should be placed on the right side of the referenced object.

C. Camera Positioning :

1. 90 degree from object i.e. top-view.

D. Image Clarity :

1. Image should contain only the package and reference object, all other should be cropped.
