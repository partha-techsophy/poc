Guidelines for taking the two pictures for machine learning training data

You will be taking two pictures. Each Picture will have both the package you would be shipping and a reference object which can be a quarter, dollar bill, business card, or standard sheet of paper (8.5” x 11.5”)

A. Background :

1. Use a plain solid background such as white, black, etc. that is a different color from your package and reference object. Try to avoid surfaces that have distinct lines such as ceramic floor tiles.

2. The goal is to have the package and reference object starkly stand out from the background.

B. Reference object :

The package should be placed on the right side of the reference object.

C. Object positioning :

90 degrees from object i.e. top-view.

D. Camera Positioning :

1. 90 degree from object i.e. top-view.

E. Picture Clarity :

The two pics should contain only the package and reference object, ensuring that any other objects are not in the two pics.
